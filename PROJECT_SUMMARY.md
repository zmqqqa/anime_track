# Anime Track 项目摘要

这是一个只服务于动漫记录的 Web 应用，基于 Next.js 14、TypeScript、Tailwind CSS 和 MySQL 构建。

## 当前保留的核心能力

- 番剧条目管理：新增、编辑、删除
- 观看状态管理：追番中、已看完、已弃坑、计划看
- 集数进度管理：支持记录观看进度并写入观看历史
- 时间线回顾：按时间查看最近看过的剧集
- 元数据补全：可从外部番剧数据源补充封面、简介、标签、声优等
- 本地鉴权：支持登录、注册、管理员权限控制

## 当前项目结构方向

- 首页是动漫总览面板
- 番剧列表页负责筛选、排序和编辑
- 番剧详情页负责条目细节维护
- API 仅围绕 anime、history、auth 三类核心能力展开

## 已移除的旧能力

- 财务与预算系统
- 购物与会员管理
- 漫画记录
- 背景设置、天气、密码、网络等小工具
- 邮件挂钩
- 系统管理面板
- 路线图和聊天会话相关结构

## 当前目标

把项目持续收敛成一个稳定、简单、专注的动漫记录工具，并逐步清理历史遗留的 lint 和类型问题。
